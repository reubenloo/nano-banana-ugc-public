# Consistent UGC Image Generation

A guide for generating consistent, realistic User-Generated Content (UGC) style character images using AI image models.

## Overview

This repository contains a framework for maintaining character identity across multiple AI-generated images while achieving a natural, "shot-on-phone" aesthetic.
